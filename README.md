# Lime+

## Description

Lime+ is a streaming platform designed to give you access to the video-on-demand content from top streamers. With Lime+, you can enjoy a wide variety of gaming content, including but not limited to action, fps, adventure and variety.

## Technologies

[TypeScript](https://www.typescriptlang.org/)
[Vite](https://vitejs.dev/)
[Tailwindcss](https://tailwindcss.com/)
[React](https://reactjs.org/)
[NodeJS](https://nodejs.org/en/)
[Mongoose](https://mongoosejs.com/)
[Fastify](https://www.fastify.io/)
